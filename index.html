Featured Work

TCS-AMWAY Super Set Tool
HTML5 · CSS3 · Vanilla JavaScript · Zero Dependencies · Single File · Production

Independently conceived, designed, and built a fully self-contained, browser-based operational toolkit that consolidates 7 distinct NOC workflow modules into a single professional-grade interface. Built to solve real pain points observed on shift: slow manual email drafting, inconsistent incident notes, missed timezone errors, and fragmented contact lookups. Live in production — actively used daily by the Command Centre team.

TOOL A
Auto JIO ISP Mail
Auto-generates professional ISP escalation emails based on outage type, ISP name, ticket number, and client details.
TOOL B
Incident Broadcast Message
Dynamically builds outage broadcast messages for all stakeholders — one click to copy.
TOOL C
Server Outage Time Calculator
Calculates precise server outage duration with custom datetime picker and auto-formats result for closure notes.
TOOL D
Timezone Converter & Live POC Finder
Live digital clock for client's primary timezone; auto-filters correct POC based on current hours and day type.
TOOL E
Automated PoC Email Generator
Generates dynamic PoC emails pre-filled with contact details, shift timings, and incident context via real-time timezone logic.
TOOL F
INCM Server Contact Lookup
Type a server name, instantly retrieve physical location, local contacts, phone numbers, and working hours.
TOOL G
Incident Closure Template Library
One-click access to 12 pre-built incident closure templates in a modal with single-click copy.
